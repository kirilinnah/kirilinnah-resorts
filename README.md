# PNG Resorts Website

This is a simple static website showcasing resorts in Papua New Guinea. The structure is as follows:

```
resorts-site/
  index.html       - homepage with hero, featured resorts
  about.html       - information about Papua New Guinea
  resorts.html     - detailed list of resorts with descriptions
  gallery.html     - photo gallery (uses placeholder images)
  contact.html     - contact form and location map
  css/
    style.css      - shared stylesheet
  js/
    script.js      - basic form validation and future interactivity
  images/          - directory for local images (currently empty)
```

## Enhancements
- Responsive grid layout for resort cards and gallery images
- Placeholder images from placeholder.com (replace with your own)
- Form validation script on contact page
- Consistent navigation across all pages

## To customize
1. Add real images to the `images/` folder and update the `src` attributes.
2. Replace the map iframe URL with an actual Google Maps embed.
3. Expand resort descriptions or add booking links.
4. Tweak styling in `css/style.css` for colors, fonts, and layout.

Enjoy building your PNG resorts site!
